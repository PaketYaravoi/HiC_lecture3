
# Вводный курс по языку программирования C# Лекция №3 "Функции: продолжение"

Добро пожаловать в репозиторий, посвященный вводному курсу по языку программирования C#. Здесь собраны проекты, выполненные в рамках изучения основ этого мощного и современного языка программирования.
Репозиторий организован по папкам, каждая из которых содержит отдельный проект, решающий конкретную задачу или ряд, простых проектов(задач).

## Структура репозитория

- **InputTask: Вводная часть, знакомство с условными видами методами **
  - Условные деления на типы методов (4 вида)
  - Написание каждого из методов
  - Вызов методов
  - Знакомство с конструкцие "for"
  - Вызов цикла в цикле в рамках метода построения таблицы умножения  

- **testTasks: Тренировочная задача для закрепления**
  - Тренировачная задача по заменен значений в строке
  - Применения цикла for 
  - Примеры ветвления if
  - Последующие вызовы метода, неоднократно, с изменением строки в тот вид , что необходим условному "пользователю"

- **selectionsort: Работа с методами. Сортировка методом выбора**
  - Определение и вызов методов
  - Параметры методов и возвращаемые значения
  - Пример рекурсивного методов

- **SelectionSortMax: Самостоятельная работа. Сортировка методом выбора**
  - Закрепление рекурсивного метода
  

## Как использовать этот репозиторий

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/ваш-аккаунт/ваш-репозиторий.git
   ```
   
2. **Открытие проекта в IDE**
   - Используйте Visual Studio или другую предпочитаемую IDE для открытия папки с проектом.

3. **Изучение и запуск примеров**
   - Перейдите в соответствующую папку и откройте проект.
   - Изучите код и запустите примеры для лучшего понимания материала.
